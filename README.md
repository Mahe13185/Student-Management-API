# 🎓 Student Management API

A RESTful Student Management System built with Spring Boot following layered architecture and enterprise development practices.

---

## 🚀 Features

- Student CRUD APIs
- Role CRUD APIs
- DTO Architecture
- Spring Data JPA
- MySQL Database
- Bean Validation
- Swagger UI
- Layered Architecture
- JPA Relationships (Many-to-One)

---

## 🛠 Tech Stack

- Java 21/26
- Spring Boot
- Spring Data JPA
- Hibernate
- MySQL
- Maven
- Lombok
- Swagger (OpenAPI)
